# Vehicle Record Management System

## Overview
A Python console application for managing vehicle records with search and statistics functionality.

## Features
- Add new vehicle records (number, owner, type)
- View all stored vehicle records
- Search vehicle by registration number
- Vehicle statistics including total count and breakdown by type
- Duplicate vehicle number detection

## Vehicle Types Supported
- Car
- Bike
- Truck

## Tools Used
- Python, Dictionaries, Lists, Functions, Input Validation
